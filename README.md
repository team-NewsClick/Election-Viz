## Getting Started

First, install all the **node_modules**

```bash
npm install
#or
yarn install
```

Secondly, run the development server:

```bash
npm run dev
# or
yarn dev
```

- Open [http://localhost:3000/elections](http://localhost:3000/elections) from the browser to see the result.
